# Music Trivia

Do you live for New Release Fridays on music platforms? Do you watch recaps of the Grammy's to find out who won various awards? Here's a simple web app that asks fun trivia questions for music lovers.

## Technologies
- HTML5
- CSS3
- JavaScript
- jQuery
- Trivia Database API
